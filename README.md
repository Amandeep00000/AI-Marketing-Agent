# AI Marketing & Automation Agent

## Overview
This project is an AI-powered marketing assistant that automates content creation using AI.

## Features
- Generates Instagram captions, ad copies, blog ideas
- Uses OpenAI GPT model
- Streamlit UI for interaction

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Use Case
Helps businesses automate marketing workflows and content generation.
